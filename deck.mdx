## Node.js

---

## História

- Node.js foi criado em 2009 por Ryan Dahl
- Originalmente era uma implementação do V8 em C++
- Em 2010, o Node.js foi lançado como software livre

---

## Principais Características

- Orientado a eventos
- Não bloqueante
- Single-threaded
- Leve e escalável
- Módulos reutilizáveis

---

## Ambiente

- Instale o Node.js no seu computador
- Crie um novo projeto com `npm init`
- Adicione pacotes com `npm install`
- Execute o código com `node nome_do_arquivo.js`
